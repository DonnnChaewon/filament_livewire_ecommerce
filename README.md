# How to run
Enter ```cmd /k code``` in the folder directory.<br>
Right click Laragon -> Run as administrator -> Yes -> Taskbar -> Show hidden icons -> Right click on the lime colored background -> Start all. After that, Laragon -> Database -> Open.<br>
Create new database with the name ```filament_livewire_ecommerce```.<br>
Open Visual Studio Code and add two new terminals.<br>
- Type ```php artisan serve``` in the first one and let it be.
- Type ```npm run dev``` in the second one and CTRL + click the ```http://localhost:8000```.
- Open new terminal and type ```php artisan add:filament-user```.<br>
Open new tab and then type ```http://localhost:8000/admin``` for the Filament login page.<br>
